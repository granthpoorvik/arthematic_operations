# arthematic_operations
this code is used to convert the list of numbers and its associated operations in string format to actual operations  in integer format  

input  formate
["32 + 698", "3801 - 2", "45 + 43", "123 + 49"]
output formate

  32         1      9999      523
+  8    - 3801    + 9999    -  49
----    ------    ------    -----
  40     -3800     19998      474
